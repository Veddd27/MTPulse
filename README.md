## What is MTPulse?
You can manage MTProto proxies by using this script.it will use the [official telegram MTProxy ](https://github.com/TelegramMessenger/MTProxy) project.
## How to Run?
Run the following script on your server (Debian or Ubuntu):
```
bash <(curl -Ls https://raw.githubusercontent.com/Erfan-XRay/MTPulse/main/MTPulse.sh)
```
